# ADN + “Genoma del Vacío” (metáfora) + Universo inspirado en Einstein (modelo didáctico)

Stack:
- Node.js (runtime)
- Next.js App Router
- TypeScript (principal)
- JavaScript (configs + helper `src/lib/exporters.js`)
- Three.js + @react-three/fiber + drei
- Zustand
- TailwindCSS

## Veracidad y metáfora
**Representación científica (didáctica):**
- ADN: doble hélice, pares A–T y C–G, backbone representado, idea general de surco mayor/menor y empaquetamiento.

**Metáfora / modelo simplificado:**
- “Genoma del vacío”: campo de ruido procedimental y partículas.
- “Einstein”: pozo gravitatorio + geodésicas aproximadas (NO simulación relativista exacta).
- “Célula”: auto-organización heurística (NO bioquímica real).

## Requisitos
- Node.js >= 20
